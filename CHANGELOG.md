# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-12-XX

### Added

- Initial release of `@gaia-tools/coeus-api-client`
- TypeScript client SDK for the Gaia astrological charting backend API
- Full API coverage for Charts, Chart Instances, and Wheels
- Type-safe API methods with comprehensive TypeScript types
- Support for both CommonJS and ES Module formats
- SSR support for Next.js applications
- `buildIndexes` utility function for efficient render data lookups
- Comprehensive type exports for all API DTOs
- Full TypeScript declarations with source maps

### Features

- **Charts API**: Create, list, get, update, delete, and duplicate chart definitions
- **Chart Instances API**: Create, list, get, update, and render chart instances
- **Wheels API**: List and get wheel configurations
- **Type Safety**: Complete TypeScript types for all API requests and responses
- **Flexible Configuration**: Support for custom Axios configuration including authentication headers
- **SSR Ready**: Works seamlessly with Next.js server-side rendering

### Documentation

- Complete README with installation, usage examples, and API documentation
- Type documentation for all exported types
- Examples for common use cases including chart creation and rendering
